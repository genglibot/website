GERA: Geometric Embedding for Efficient Point Registration Analysis
Geng Li, Haozhi Cao, Mingyang Liu, Shenghai Yuan, Jianfei Yang

Project Page / arXiv

We propose a novel point cloud registration network that uses a pure MLP architecture with offline-constructed geometric encoding, eliminating complex modules like KPConv and Transformers, thereby significantly reducing computational demands and improving generalization and stability for resource-constrained applications.
